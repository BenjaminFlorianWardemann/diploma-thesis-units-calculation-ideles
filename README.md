# Zur Einheitenberechnung mittels Idelen

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.18519141.svg)](https://doi.org/10.5281/zenodo.18519141)
[![License: CC BY 4.0](https://img.shields.io/badge/License-CC%20BY%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by/4.0/)

This repository contains the source code and the digital version of my diploma thesis in Mathematics, originally submitted to **TU Berlin** in 2011.

The work focuses on the computation of $S$-units in global fields, i.e. algebraic number fields and function fields, using ideles, specifically investigating algorithms within the framework of the Magma computer algebra system.

The full thesis is available via Zenodo: [https://doi.org/10.5281/zenodo.18519141](https://doi.org/10.5281/zenodo.18519141)
